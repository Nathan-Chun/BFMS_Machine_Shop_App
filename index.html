Welcome to the BFMS Machine Shop Website

The BFMS machine shope is part of the USC Viterbi Baum Family Makerspace. Our machine shop is equipped with 3-axis mills, lathes, a waterjet, and much more. More information about our machines can be found on the BFMS website.

If you'd like to submit a work order, please complete this form.

Various calculators are available on this page to help you determine the speeds and feeds for various tools. This is especially useful when programming toolpaths in MasterCAM. The values are all based on charts published by HASS Automation, Inc online.
